# typescript_project

## Setup

1. **Clone this repository**
2. **Open Terminal in VSCode** (Command Prompt)
3. **Install dependencies**

```sh
npm install
```

## Test Cases

This project includes unit tests for the `merge` function, such as:

- Merging sorted arrays
- Handling empty arrays
- Handling duplicate values
- Handling negative numbers

## Run Unit Tests

Open your terminal and run:

```sh
npm test
```
